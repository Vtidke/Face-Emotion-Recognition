# Facial Emotion Recognition using CNN (MobileNetV2)

This project detects human emotions such as happy, sad, angry, and neutral using deep learning (MobileNetV2) and OpenCV.

## 🧠 Features
- Trains a CNN model using MobileNetV2.
- Real-time emotion detection using OpenCV.
- Supports multiple facial expressions.
